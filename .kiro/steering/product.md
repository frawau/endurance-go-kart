# GoKartRace - Product Overview

GoKartRace is a comprehensive Django-based management system for endurance go-kart races and championships. It provides real-time race control, penalty management, and hardware integration for professional race management.

## Core Features

- **Championship & Round Management**: Multi-round championships with flexible race configuration
- **Team & Driver Management**: Complete participant registration and team formation
- **Real-time Race Control**: Live race monitoring with start/pause/resume controls and WebSocket updates
- **Advanced Penalty System**: Multiple penalty types (Stop & Go, Laps, Post Race) with hardware integration
- **Hardware Integration**: Raspberry Pi-based penalty stations with physical controls and displays
- **API & Mobile Support**: RESTful API with QR code integration for mobile workflows

## Target Users

- Race directors and officials
- Team managers and drivers
- Event organizers
- Hardware station operators

## Key Workflows

1. **Pre-Race**: Championship setup → Team registration → Driver assignment → Round configuration
2. **Race Day**: Pre-race checks → Race start → Live monitoring → Penalty management → Race end
3. **Post-Race**: Automatic penalty calculation → Results processing → Championship standings

The system excels at race management and penalty administration while supporting integration with external timing systems.
